Usage

curl -X POST -H 'Content-Type: application/json' -i http://localhost:8080/api/v1/report --data '{
  "filterBy": [
    {
      "field": "DATASOURCE",
      "values": ["Facebook Ads"]
    },
    {
      "field": "CAMPAIGN",
      "values": ["SN_Essen", "SN_Brand Produkte"]
    }
  ],
  "groupBy": "CAMPAIGN",
  "orderBy": "CAMPAIGN",
  "orderDir": "DESC"
}
'
Note that depending on grouping, some fields might be unavailable.
